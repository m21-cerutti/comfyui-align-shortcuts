# Contribution Guide

This project welcomes contributions! If you find a bug or want to add a feature, please follow these steps:

1. Fork the repository and create a new branch (`git checkout -b feature/amazing-feature`).
2. Make your changes (e.g., adding code, fixing bugs).
3. Commit them into the branch (`git commit -m 'feat: Added awesome functionality'`).
4. Push to the branch (`git push origin feature/amazing-feature`).
5. Open a Pull Request!

## Contribution Rules

Please adhere to these guidelines when contributing:

- Maintain code style consistency (check neighboring files for patterns).
- All new features should be accompanied by corresponding documentation, even greater, tests.
